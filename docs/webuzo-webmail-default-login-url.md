# Webuzo Webmail Default Login URL

If you are using a web server's DNS with its nameserver, then the following URL will work to log in to Webmail.

1) https://mail.domain.com
2) https://domain.com:2003/mail
3) https://domain.com/webmail


!!! note
    There is no specific port like cPanel
