Navigate to `Home > Settings > Panel Config` and increase `Webuzo PHP Max Post Size` and increase `Webuzo PHP Max Upload Size` and also increase `Webuzo Service Client Max Body Size`.


![webuzo-increase-sql-upload-size](https://github.com/cornQ/webuzo-server-setup-almalinux9/blob/main/img/webuzo-increase-sql-upload-size.png "webuzo-increase-sql-upload-size")


> Note:  `Webuzo PHP Max Post Size` is mostly greater than the `Webuzo PHP Max Upload Size`.


### This method doesn't always work, especially for a huge database. Use mysqldump instead of this method.
