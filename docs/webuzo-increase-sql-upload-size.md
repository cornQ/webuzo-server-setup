Navigate to `Home > Settings > Panel Config` and increase __Webuzo PHP Max Post Size__ and increase __Webuzo PHP Max Upload Size__ and also increase __Webuzo Service Client Max Body Size__.


![webuzo-increase-sql-upload-size](img/webuzo-increase-sql-upload-size.png "webuzo-increase-sql-upload-size")


!!! info
    __Webuzo PHP Max Post Size__ is mostly greater than the __Webuzo PHP Max Upload Size__.


!!! tip
    This method doesn't always work, especially for a huge database. Use __mysqldump__ instead of this method.
